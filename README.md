# projek-uas-web
dalam pembuatan website Pesona Purwakarta seperti halaman beranda, halaman wisata, halaman kuliner, dan halaman pemesanan tiket, pada bagian css dan html menggunakan ai sebagai referensi dalam pemilihan font, pembuatan komponen card, dan pembuatan efek hover.
dalam pembuatan website wisata purwakarta ini ada peran dari sebuah tools yaitu AI yang mana AI ini membantu kami di beberapa bagian. salah satu penggunaannya ada pada beberapa file, dalam file js penguna meminta untuk mencari bagaimana cara membuat dark mode pada suatu fitur website ini pada file darkmode.js .
